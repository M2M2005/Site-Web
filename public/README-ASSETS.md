# 📁 GUIDE - Où placer vos fichiers

Ce fichier vous guide pour savoir où placer toutes vos images et documents.

## 📄 Documents (dossier `/docs`)

Placez les fichiers suivants dans `portfolio/public/docs/` :
- `CV_BONS.pdf` - Votre CV téléchargeable
- `Rapport_Stage_BONS_Orchestra_TravelSoft.pdf` - Rapport de stage Orchestra
